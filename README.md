# Agenda Wizard Lite

A simple and modern To-Do List web app built with React, Vite, TypeScript, and TailwindCSS.

## Features
✅ Add, edit, delete tasks  
✅ Mark tasks as completed  
✅ Responsive and clean UI  

## Tech Stack
- React + TypeScript
- Vite
- TailwindCSS
- Radix UI components

## Setup

```bash
npm install
npm run dev
